Lightweight and easy to use plugin which displays a customizable banner.

== Description ==

Given the current corona epidemic situation, a lot of website and business owners find themselves in need of communicating how this affects their business, readers, clients or customers. This simple plugin will display a customizable banner in the header of your site. You'll be able to add your own content including a link to a page where you can  share more information if needed.

Check out the plugin in action at the <a href="https://www.coronawp.com/">plugin website</a>.

<strong>In the plugin settings you can:</strong>
 
* Add your own title and excerpt
* Add your own URL and link text
* Option to open link in new window
* Change colors
* Set content alignment
* Set banner to be sticky
* Set banner position to top/bottom
* Turn the banner on/off

= Instruction video =
https://www.youtube.com/watch?v=RxImYpSxMZU

= How to use it =
After installing the plugin, you will find a new menu item in the wp-admin panel called CoronaWP. Define your settings and check the Active checkbox, and you're all set.
  
== Installation ==
  
1. Upload plugin folder to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Change the settings accordingly to your needs
 
== Frequently Asked Questions ==

= Can it be used to display any info?

Yes. You can use it to display any info you'll like. You're not limited to use it for COVID-19 information only, as you set your own text and link.

= Do I need to have coding knowledge to use this plugin? =
 
No. It's plug and play.

= Can I set my own custom colors?

If you want to set other colors than the supplied color themes, you can use the Wordpress Customizer to add your own CSS. The ID of the banner is <strong>coronawp</strong>, and the button class is <strong>cwp_button</strong>.

= Can I set my own text? =
 
Yes. The you can add your own custom text, and set options like color theme etc.

= Do you have a demo site? =

Yes. Go visit our site <a href="https://www.coronawp.com">CoronaWP.com</a> for example banners.

= Is CoronaWP compatible with Elementor page builder?

Yes. It works very well with Elementor. 

= Is the plugin GDPR compliant?

Yes. CoronaWP sets no cookies or stores any visitor data.
